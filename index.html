Example Displayer

A Displayer is rendered by combining an .html and .js source. The steps are as follows:

  1. Ask the client library for a displayer for an InterMine object.
  2. The library loads the data part (JSON), templates (html) from the server while providing a callback for when presenter ("how should data be rendered) is returned from the server.
  3. The presenter is initialized with a bunch of options like imObj, displayer name, data payload etc.
  4. underscore.js templating function is used to populate templates from the server with the payload.
  5. As presenters are arbitrary objects with functions, any kind of behavior can be encompassed.
